개념 지도 — KSEPT · LLM로 만들기

이 사이트는 7개 모듈 튜토리얼의 개념 노트 모음입니다. 각 모듈은 ① 한 줄 요약 → ② 비유 → ③ 이 앱에서는 → ④ 더 깊이 → ⑤ 미니 퀴즈 형식으로, 비유부터 차근차근 풀어요. 아래 흐름을 따라가세요.

42에서 이미 해봤어요. 새 개념이라기보다, 당신이 만든 프로젝트의 LLM 버전입니다. ft_inception의 nginx + 여러 컨테이너, ft_transcendence의 SPA + 백엔드 + 로그인 경험이 여기 두 서버 + 프록시, React fetch + stateless 이야기로 그대로 이어집니다.

→ 커밋마다 한 일은 변경 이력에서.

전체 흐름

Setup에서 출발해 한 모듈씩 쌓아 올려 Workshop(빌드 데이)로 끝납니다.

flowchart TB M1["1 · Setup"] M2["2 · Foundations"] M3["3 · Tools & Structure"] M4["4 · Context / RAG"] M5["5 · Agents"] M6["6 · Production"] M7["7 · Workshop"] M1 --> M2 --> M3 --> M4 --> M5 --> M6 --> M7

모든 모듈에 내용이 채워졌고, 각 섹션에 원본 슬라이드 번호(예: 슬라이드 17)와 영어 제목을 붙여 강의 덱과 맞췄습니다.
